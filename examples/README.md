# CS2 Server Manager Examples

This directory contains example scripts and code for using the CS2 Server Manager.

## Available Examples

- [examples.py](examples.py) - API usage examples demonstrating how to interact with the CS2 Server Manager API

## Running the Examples

### examples.py

This script demonstrates various API endpoints and usage patterns:

```bash
python examples/examples.py
```

**Note:** Make sure the API server is running first:

```bash
python main.py
```

The examples script will output sample API requests and curl commands that you can use to interact with the server.

## API Documentation

For interactive API documentation, visit:
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

For more information, see the [main README](../README.md)
