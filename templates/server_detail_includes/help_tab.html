Help & Troubleshooting

This section provides common troubleshooting guides and fixes for CS2 server issues.

KZ Server Environment Fix (Ubuntu 24.04+)
Important Notice

Please log in to your server as root user to perform this operation.

Due to requiring high system privileges, the system will not complete this operation for you using the server user account.

Problem Description

On Ubuntu 24.04+ systems, KZ server plugins may fail to load due to missing libssl1.1 library.

Solution Steps

Follow these steps to fix the plugin loading issue:

  1. Download the libssl1.1 package:
    wget https://security.ubuntu.com/ubuntu/pool/main/o/openssl/libssl1.1_1.1.1f-1ubuntu2.24_amd64.deb
  2. Install the package:
    sudo dpkg -i libssl1.1_1.1.1f-1ubuntu2.24_amd64.deb
  3. Restart your CS2 server

    After installing the package, restart your CS2 server to apply the changes.

Security Warning

These commands must be executed manually on your server. The server manager will not automatically execute these commands for security reasons and due to the need for elevated privileges.
